# 金鑫大利合作需求評估書 — 教務系統 03/04 調整版

## 1. 排課
- (優化) 新增排課：團班/安親/個指排課模型已做出，需加入學生名單到班級、快速了解班級學生
  - 團班/安親新增課程：提供學生名單列表讓操作方勾選
  - 個指新增課程：每個個指班級僅限一位學生，同時段可安排多位學生，可配置不同老師+同步按鈕
- (優化) 排課一覽介面：行事曆模式（週/日/月），右側行事曆+左側排課列表（全部/團班/安親/個指），顯示上課中/已下課/已結束
- (新增) 每項課程提供編輯功能
- (新增) 安排補課模型：補課按鈕→選現有補課課程或新增→日期時間+老師+課程+學生→通知家長

## 2. 學生管理
- (討論) 學生與家長綁定機制：家長如何知道孩子身份並收到通知
- (優化) 學生年級：透過出生日期自動計算年級（台灣體制）

## 3. 出缺勤
- (優化) 學生端：點名模式新增「未到(曠課)」選項，通知家長
- (優化) 師資端：比照學生端，曠職通知老師
- (優化) 曠課/請假學生：安排補課功能

- (優化) 個指：個別進度+個別作業
- (優化) 小叮嚀+考試成績錄入+照片上傳
