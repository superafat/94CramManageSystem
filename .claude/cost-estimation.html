⚠ 注意事項與假設說明

  • Cloud SQL 升級至 4 vCPU / 16GB RAM 以承載 21,900 位使用者的並發查詢
  • Cloud Run 採 scale-to-zero，閒置不計費；尖峰時段自動擴展，費用依實際使用量
  • LINE 是最大成本（73.6%）：善用 Reply Message（免費）+ Flex Message 可大幅降低推播量
  • Cloud SQL HA 為選配但強烈建議；不開 HA 可省約 NT$8,960/月
  • 方案 A vs B：各校自營 LINE 每校需 NT$4,000，平台統一僅 NT$730（省 82%）
  • 未含項目：人事成本、客服成本、開發維護成本、CDN 加速、簡訊驗證費
  • 匯率波動：USD/NTD 以 1:32 估算，實際依即時匯率為準
  • 以上為 GCP 公開定價估算，實際帳單可能因 Sustained Use Discount 或 CUD 承諾折扣 降低 20–30%
  • 超過 200 校建議加入 Cloud SQL Read Replica（+~NT$9,000/月）與 Cloud CDN 提升效能
